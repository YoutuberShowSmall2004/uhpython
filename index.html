Оператор условия if (пример):
if a == 2:
    print ('Верно')
else:
    print ('Неверно')
Если число a будет равно двум, то будет написано "верно", а "else" означает, что если условие не выполнится, то будет написано "Неверно".
Перед "print" 4 отступа.
Цикл for (пример):
for i in range (1, 1000):
print (i)
Программа выводит числа от 1 до 1000.
